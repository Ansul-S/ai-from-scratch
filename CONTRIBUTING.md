# Contributing to AI From Scratch

Thank you for contributing! This repo exists to help people learn.

## Ways to contribute
- Fix a bug in an existing implementation
- Improve an explanation in a README
- Add interview questions you've encountered
- Propose a new project

## Project standards
- README.md with all required sections
- notebook.ipynb with clear markdown cells
- solution.py with clean, commented code
- tests/test_solution.py with at least 3 unit tests
- Core rule: implementations use only NumPy

## Submitting a PR
1. Fork and create a branch: `git checkout -b feature/your-project-name`
2. Commit: `git commit -m "Add: project name"`
3. Push and open a PR against main
